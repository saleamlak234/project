## 🔍 Accessing Transaction Files

### Step 1: Navigate to Transactions
1. Login to your admin account
2. Go to **Admin Dashboard** → **Transactions**
3. Find the transaction you want to review
4. Click the **"View"** button in the Actions column

### Step 2: View Transaction Details
- The transaction preview page will open
- Scroll down to the **"Receipt"** section on the right side
- You'll see the uploaded file with preview options

## 🖼️ File Preview Features

### Image Files (JPG, PNG, GIF, WebP, BMP)
- **Thumbnail Display**: Small preview image shown automatically
- **Click to Enlarge**: Click on the thumbnail to open full preview
- **Hover Effects**: Visual feedback when hovering over images

### PDF Files
- **PDF Icon**: Shows a PDF icon with file information
- **View PDF Button**: Click to open PDF in new tab
- **Download Option**: Direct download available

### Other Files
- **File Icon**: Generic file icon for unsupported preview types
- **View File Button**: Opens file in new tab if browser supports it
- **Download Option**: Always available for all file types

---

## 📁 Supported File Types

### ✅ **Fully Supported (with Preview)**
- **Images**: JPG, JPEG, PNG, GIF, WebP, BMP
- **Documents**: PDF

### ⚠️ **Partially Supported (download only)**
- **Documents**: DOC, DOCX, XLS, XLSX
- **Archives**: ZIP, RAR
- **Other**: Any file type (download available)

### 📏 **File Size Limits**
- Maximum upload size: **10MB**
- Recommended size: **5MB or less** for optimal performance

## 🎛️ Image Preview Controls

### Opening Image Preview
1. Click on the receipt thumbnail in the transaction details
2. The full-screen image preview will open
3. Use the control buttons in the top-right corner

### Available Controls

#### 🔍 **Zoom Controls**
- **Zoom In (+)**: Increase image size up to 500%
- **Zoom Out (-)**: Decrease image size down to 25%
- **Current Zoom**: Displayed in top-left corner

#### 🔄 **Rotation**
- **Rotate Button**: Rotates image 90° clockwise
- **Keyboard Shortcut**: Press `R` key

#### 📐 **Pan and Move**
- **Drag to Pan**: When zoomed in, drag the image to move around
- **Auto-Center**: Image centers automatically when zoom changes

#### 🖥️ **Display Options**
- **Fullscreen Toggle**: Expand to full browser window
- **Reset View**: Return to original size and position

#### 💾 **Download Options**
- **Download Button**: Save image to your computer
- **Right-Click**: Use browser's "Save Image As" option

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `ESC` | Close preview |
| `+` or `=` | Zoom in |
| `-` | Zoom out |
| `R` | Rotate 90° |

## 📥 Download Options

### Method 1: Preview Download
1. Open the image/file preview
2. Click the **Download** button in the controls
3. File will be saved with a descriptive name

### Method 2: Direct Download
1. In the transaction details page
2. Click the **Download** link next to the file info
3. File downloads immediately

### Method 3: Right-Click Download
1. Right-click on any image thumbnail
2. Select **"Save Image As"** from the context menu
3. Choose your download location

## 🔧 File Management Features

### File Information Display
- **File Type**: Automatically detected and displayed
- **File Size**: Shown when available

### Quality Indicators
- **✅ High Quality**: Clear, readable images
- **⚠️ Medium Quality**: Acceptable but may need zoom
- **❌ Poor Quality**: Difficult to read, may need re-upload

### Error Handling
- **Failed to Load**: Shows error icon if file is corrupted
- **Missing File**: Indicates if file was deleted from server
- **Access Denied**: Shows if file permissions are incorrect

## 🚨 Troubleshooting

### Common Issues and Solutions

#### **Image Won't Load**
- **Check Internet Connection**: Ensure stable connection
- **Refresh Page**: Press F5 or Ctrl+R to reload
- **Clear Browser Cache**: Clear cache and cookies
- **Try Different Browser**: Test with Chrome, Firefox, or Edge

#### **Preview is Slow**
- **Large File Size**: Files over 5MB may load slowly
- **Network Speed**: Check your internet connection
- **Browser Performance**: Close other tabs to free up memory

#### **Can't Download File**
- **Popup Blocker**: Disable popup blocker for the site
- **Browser Settings**: Check download permissions
- **Antivirus Software**: Temporarily disable if blocking downloads

#### **Image Quality is Poor**
- **Zoom Feature**: Use zoom controls to enlarge image
- **Original Resolution**: Download original file for best quality
- **Request Re-upload**: Ask user to upload higher quality image

## 💡 Best Practices

### For Reviewing Receipts
1. **Always Preview**: Check image quality before approving
2. **Verify Details**: Ensure receipt matches transaction amount
3. **Check Authenticity**: Look for signs of tampering or editing
4. **Document Issues**: Note any problems in rejection reasons

### For File Management
1. **Regular Cleanup**: Archive old transaction files periodically
2. **Backup Important Files**: Download critical receipts for records
3. **Monitor Storage**: Keep track of server storage usage
4. **Update Browser**: Use latest browser version for best performance
